### **🎣 오낚끝 패치노트**

> 버전 포맷은 mm.d.num입니다. 예를 들어 10월 9일의 첫 번째 패치라면, v10.9.1로 기입됩니다.

\
**v10.9.1** <sub>[minor]</sub>
```
- 패치노트 추가
- 오낚끝 문구 변경 (오늘 만난 게 / 오늘 진행한 미니게임 → 만난 게 / 만난 미니게임)
- 총 생직 시간에 분 포맷이 적용되었습니다. (ex. 총 610분 → 총 610분 [10시간 10분])
```

\
**v10.10.1** <sub>[minor]</sub>
```
- 패치노트가 업데이트되었다면 [NEW] 표시가 붙습니다. 확인하였다면 사라집니다
- 한 로그 파일 내에 자정이 넘어갈 시 세션 인식을 못 하는 문제를 해결했습니다
- 구글 검색이 가능해졌습니다
```
